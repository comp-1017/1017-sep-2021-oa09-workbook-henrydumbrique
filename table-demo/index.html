Table Demo

A table is a way of creating a structured grid display of data.The paired tags(open/close tags) we will be interested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the header row of the table
  • <tbody> : identifies the data detail body of the table
  • <tfoot> : identifies the footer row of the table
  • <tr> : identifies a row on the table
  • <th> : identifies the cell on the header for the column title
  • <td> : identifies the cell on the detail row or footer to hold the content(pics,info)

some tags have parameter. The border parameter of the table tag applies a border around your table. The colspan parameter and the rowspan parameter are used on the <td> tab to allow you to merge columns on a single row or column cells across rows.
